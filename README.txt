parse.sh 
- Replace name of file
- Output files are named after the nodes (ex: amaranth1, hulk2, etc.,)
- Run it as 
    - sh parse.sh
    
    
    
time.sh
- takes the time as arguments
- first three correspond to time on amaranths
- next three to hulk2
- last three to hulk[3-4]

  - sh time.sh 09:16:40 09:23:21 09:30:09 01:16:40 01:23:21 01:30:09 02:16:40 02:23:21 02:30:09
